# Donasi React (Tailwind + TypeScript) - Mockup Reconstruction

## Apa yang ada di paket ini
- Vite + React + TypeScript skeleton
- Tailwind CSS with `tailwind.config.cjs` and `postcss.config.cjs`
- Komponen di `src/components/*.tsx` yang sudah memakai Tailwind utility classes
- Entry: `src/main.tsx`, `src/App.tsx`, `index.html`

## Cara pakai (Windows)
1. Extract zip ke folder project (misal: `C:\Users\You\Desktop\donasi`)
2. Buka terminal di folder project.
3. Jalankan:
```
npm install
npm run dev
```
4. Buka `http://localhost:5173`

> Catatan: package.json di-include di zip. Pastikan Node 16+ dan npm terbaru.

Jika mau saya bisa:
- Integrasikan ke repo Vite yang sudah ada (patch),
- Tambahkan plugin Tailwind forms/typography,
- Menghubungkan komponen ke Supabase (auth + data).
